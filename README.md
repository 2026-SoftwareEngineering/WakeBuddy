# WakeBuddy

WakeBuddy is a shared alarm app that allows connected users to create, modify, and delete alarms together.


## 프로젝트 개요

WakeBuddy는 연결된 사용자끼리 알람을 생성하고 수정할 수 있는 공유 알람 앱이다.
본 프로젝트는 소프트웨어 공학 수업의 바이브코딩 과제로, 요구사항 분석에서 품질 관리에 이르기까지의 개발 프로세스를 적용하여 AI 도구로 소프트웨어를 개발하는 것을 목표로 한다.


## 주요 기능

### 1. 내 알람 관리

- 내 알람 조회
- 내 알람 생성
- 내 알람 수정
- 내 알람 삭제

### 3. 친구 알람 관리

- 친구 알람 조회
- 친구 알람 생성
- 친구 알람 수정
- 친구 알람 삭제

### 3. 친구 관리

- 친구 목록 조회
- 친구 추가


## 기능 범위

본 프로젝트는 공유 알람 앱의 핵심 기능에 집중하기 위해 내 알람 관리, 친구 알람 관리, 친구 관리기능으로 범위를 제한한다.
사용자는 연결된 친구에게만 알람을 생성할 수 있으며, 자신이 생성한 친구 알람을 수정하거나 삭제할 수 있다.
